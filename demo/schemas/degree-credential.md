# Degree Credential

This credential represents a university degree certificate issued to a student.

## Credential Subject Properties

### degree
- **Type**: string
- **Description**: The type of degree (e.g., "Bachelor of Science", "Master of Arts")

### name
- **Type**: string
- **Description**: The full name of the degree recipient

### university
- **Type**: string
- **Description**: The name of the university that issued the degree

### graduationYear
- **Type**: integer
- **Description**: The year of graduation (e.g., 2023)
